# Vibe Coding — учебный проект

## Что это

Тренировочный проект для изучения веб-разработки и смежных технологий.
Здесь отрабатываются навыки создания страниц, ботов и работы с API.

## Для кого

Для Станислава Нещерета — риелтора, который осваивает разработку
с помощью Claude Code.

## Что уже есть

### vizitka.html — личная визитка риелтора
- Тёмный дизайн в золотых тонах
- Имя, специализация, статистика (10+ лет, 500+ сделок)
- Кнопки соцсетей: Telegram, телефон, WhatsApp, Instagram, VK
- CTA-кнопка "Написать мне" → @nestas85
- Анимации: частицы, вращающееся кольцо аватара, 3D-наклон карточки

### landing.html — лендинг TaskFlow
- Тёмный дизайн в фиолетово-неоновых тонах
- Hero-секция с анимированными заголовками
- Метрики с анимацией счётчиков при скролле
- Блок преимуществ: умные приоритеты, экономия времени, команда
- Секция целевой аудитории: фрилансеры, предприниматели, студенты, менеджеры
- Финальный CTA и футер

## Что планируется

- Телеграм-бот для риелторов — поиск данных по ЖК (застройщик, сроки, цены)
  через API наш.дом.рф / apicrafter.ru
- Возможно, бот будет вынесен в отдельный проект

## Стиль дизайна

Оба компонента используют современный **dark UI**:

| | Визитка | Лендинг |
|---|---|---|
| Фон | Чёрный (#0a0a0a) | Тёмно-синий (#060610) |
| Акцент | Золото (#c8a96e) | Фиолетовый (#7c6fff) |
| Стиль | Премиальная карточка | SaaS-лендинг |
| Анимации | Частицы, glow, spin | Orbs, fadeUp, scroll reveal |

Общие принципы: градиентные тексты, стеклянные карточки, плавные hover-эффекты.

## Технологии

- Чистый HTML + CSS + JavaScript (без фреймворков)
- Git + GitHub для хранения кода
- Claude Code для разработки

## Статус

Проект в активной разработке. Домена пока нет.
