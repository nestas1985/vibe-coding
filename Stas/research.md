# Исследование: Личный AI-ассистент в Telegram

**Дата:** 04.06.2026  
**Цель:** Создать Telegram-бота, который утром присылает план дня, вечером спрашивает как прошёл день, синхронизируется с задачником/календарём, работает 24/7 без включённого компьютера.

---

## Как это работает (общая схема)

```
[Todoist/Google Calendar]
         |
    [scheduler.py]  ← APScheduler (запускает задачи по времени)
    07:00 → читает задачи → генерирует текст через AI → шлёт в Telegram
    21:00 → спрашивает "как прошёл день?" → ждёт ответ → сохраняет в БД
         |
    [ai.py]  ← Gemini или GPT-4o mini (понимает русский, пишет ответы)
         |
    [SQLite]  ← хранит задачи, историю, заметки
         |
  [Telegram-бот]  ←→  [Станислав]
```

Всё это крутится на сервере в интернете (Railway.app или Oracle Cloud) — компьютер выключать можно.

---

## 1. Telegram-бот: что выбрать

**Лучшая библиотека:** `aiogram 3` (Python)

- Полностью асинхронная, современная
- Встроенная система диалогов (FSM) — бот задаёт вопрос и ждёт ответа
- Активное сообщество, много примеров на русском

**Расписание сообщений** — через `APScheduler`:
```python
scheduler.add_job(send_morning, "cron", hour=7, minute=0)
scheduler.add_job(send_evening, "cron", hour=21, minute=0)
```

**Диалоги** — бот задаёт вопрос, устанавливает "состояние ожидания", обрабатывает ответ. Это называется FSM (Finite State Machine).

## 2. Хостинг: где держать бота 24/7

| Платформа | Всегда включён | Стоимость | Сложность |
|---|---|---|---|
| **Oracle Cloud** | ✅ Да | 🆓 Бесплатно навсегда | Сложно получить (слоты разбирают) |
| **Railway.app** | ✅ Да | $5 бесплатно → потом ~$0.30/мес | Очень легко |
| **Render.com** | ❌ Засыпает | Бесплатно только веб, бот — от $7/мес | Не подходит |
| **Koyeb** | ⚠️ Засыпает если нет трафика | Бесплатно | Рискованно |
| **GitHub Actions** | ❌ Только по расписанию | 🆓 Бесплатно | Только отправка, без приёма |

**Рекомендация для старта:** Railway.app — запускается за 10 минут, не засыпает, $5 хватит на несколько месяцев, потом копейки.

**Идеал (если получить):** Oracle Cloud Free Tier — настоящий виртуальный сервер, бесплатно навсегда.

## 3. Задачник/Календарь: куда сохранять задачи

| Сервис | Простота | Бесплатный тариф | Вывод |
|---|---|---|---|
| **Todoist** | ⭐⭐⭐ Очень просто | 5 проектов, 300 задач | Лучший выбор |
| **Notion** | ⭐⭐ Средне | Хороший (3 запроса/сек) | Хороший вариант |
| **Google Tasks** | ⭐ Сложно (OAuth) | Щедрый | Можно, но сложно |
| **Google Calendar** | ⭐ Сложно (OAuth) | Щедрый | Избыточно для задач |

**Рекомендация:** Todoist — один токен, простое API, Python-библиотека `todoist-api-python` работает из коробки. Для Google Calendar — позже, если понадобится.

## 4. AI-движок: что генерирует тексты

Для личного бота (10-20 сообщений в день):

| Провайдер | Модель | Цена/месяц | Русский язык |
|---|---|---|---|
| **Google Gemini** | Gemini 2.5 Flash | 🆓 Бесплатно (1500 req/день) | Отлично |
| **Groq** | Llama 4 | 🆓 Бесплатно (1000 req/день) | Хорошо |
| **OpenAI** | GPT-4o mini | ~$0.05/мес | Отлично |
| **Anthropic** | Claude Haiku 4.5 | ~$0.20/мес | Отлично |

**Рекомендация для старта:** Gemini 2.5 Flash — бесплатно, русский понимает хорошо, API простое.

**Память бота:** хранить последние 20 сообщений в JSON-файле или SQLite, каждый раз добавлять в запрос как "история разговора". Никаких векторных баз данных — это на потом.
