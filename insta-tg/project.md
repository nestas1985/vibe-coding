# InstaTG — проектная документация

## Что это

Полностью автоматизированная система: AI создаёт контент → публикует в Instagram → ведёт аудиторию в Telegram AI-агентов → монетизация через подписки.

Человек подписывается на Instagram, получает пользу, переходит в Telegram-бота, начинает им пользоваться. Всё — без ручного труда со стороны владельца.

---

## Для кого

Обычные люди, которые хотят улучшить свою жизнь — бросить курить, выстроить привычки, пить больше воды, лучше спать. Не хотят качать приложения и регистрироваться — хотят просто начать прямо сейчас.

---

## Какую проблему решает

- Приложения для привычек сложные и про них забывают
- Полезный контент в Instagram не даёт реального инструмента — только лайки
- Создавать контент вручную каждый день невозможно для одного человека

**Решение:** AI сам создаёт и постит контент в Instagram. Каждый пост ведёт в конкретного Telegram-бота — готовый инструмент, который работает каждый день.

---

## Как работает схема (полная автоматизация)

```
[AI-агент контента]
        ↓
Генерирует карусель / Reels (текст + картинки)
        ↓
Автоматически публикует в Instagram
        ↓
Ссылка в посте / bio → конкретный Telegram-бот
        ↓
Пользователь запускает AI-агента в Telegram
        ↓
Бесплатный базовый доступ → платная подписка
        ↓
Довольный пользователь → подписывается на Instagram → рассказывает друзьям
        ↓
Аккаунт растёт, трафик растёт, выручка растёт
```

---

## Два главных блока

### Блок 1 — Instagram (автоматический постинг)

- AI генерирует карусели и Reels по контент-плану
- Публикация через Instagram Graph API или сторонний сервис
- Каждый пост заточен под конкретный бот (трекер привычек, отказ от курения и т.д.)
- В каждом посте — ссылка или CTA ведут в нужного бота

### Блок 2 — Telegram AI-агенты

- Каждый бот — отдельный инструмент под задачу
- Ежедневная работа: напоминания, отчёты, AI-поддержка
- Freemium: базово бесплатно, расширенный функционал — платно

## Первый бот: Трекер привычек

- Пользователь выбирает привычки (бег, вода, сон, своё)
- Каждый день: напоминание → отметка выполнения (2 тапа)
- AI поддерживает, даёт советы, не даёт бросить
- Стрики, достижения, недельный отчёт
- Freemium: 3 привычки бесплатно, безлимит + аналитика — платно

---

## Монетизация

- Freemium: бесплатный базовый доступ для всех
- Подписка: ~399₽/мес через Telegram Stars
- Цель: 20–50 платящих на старте = 8 000–20 000₽/мес

---

## Чего НЕ будет на старте

- Нескольких ботов одновременно — начинаем с одного
- Платёжных систем кроме Telegram Stars
- Интеграций с приложениями и трекерами
- Сложной аналитики и дашбордов

## Технический стек

### Telegram-боты
- Python + aiogram 3.x
- Claude API (Haiku для простых задач, Sonnet для AI-агентов)
- SQLite → PostgreSQL при росте
- APScheduler для напоминаний
- Хостинг: Hetzner VPS (~€4.5/мес)

### Instagram-автоматизация
- Генерация контента: Claude API (тексты) + DALL-E / Midjourney (картинки)
- Создание каруселей: Canva API или Python (Pillow + шаблоны)
- Публикация: Instagram Graph API (официальный) или Publer / Buffer
- Планировщик: n8n или Zapier или свой Python-скрипт

## План действий по Instagram-автоматизации

### Этап 1 — Старт (сейчас, 1–4 недели)
1. Завести бизнес-аккаунт Instagram
2. Привязать к Facebook-странице (обязательно для Graph API)
3. Зарегистрироваться как Meta Developer, создать приложение
4. Первые посты публиковать через **Publer** или **Buffer** — есть бесплатный триал
5. Аккаунт должен прогреться: новые аккаунты банят за автоматизацию особенно быстро

### Этап 2 — Прогрев (1–2 месяца)
- Публикации через Publer с расписанием (полуавтомат)
- AI уже генерирует тексты и картинки, человек только нажимает "опубликовать"
- Параллельно ждём одобрения Meta на расширенный доступ к Graph API

### Этап 3 — Полная автоматизация (после прогрева)
- Переход на официальный Instagram Graph API
- Свой Python-скрипт: генерирует контент → собирает карусель → публикует по расписанию
- Полностью без участия человека

### Варианты публикации и риски

| Способ | Плюсы | Минусы | Риск бана |
|--------|-------|--------|-----------|
| Publer / Buffer | Просто, всё умеет | Платно $15–40/мес | Низкий |
| Instagram Graph API | Бесплатно, официально | Долгое одобрение, Reels нестабильны | Низкий |
| instagrapi (неофициально) | Умеет всё, без одобрения | Против правил Instagram | Высокий для нового аккаунта |

**Вывод:** старт через Publer → при 500+ подписчиках переход на Graph API.
