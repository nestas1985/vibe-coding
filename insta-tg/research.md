## 1. Конкуренты

### 1. Fabulous (приложение)
**Ссылка:** fabulous.app
**Что умеет:** AI-коуч по привычкам и ритуалам. Утренние/вечерние рутины, трекинг, push-уведомления, медитации.
**Хорошо:** Красивый дизайн, сильная геймификация, научная база (поведенческая психология).
**Плохо:** Только приложение, нет Telegram. Дорого — $30–50/год. Нет русскоязычного контента.
**Аудитория:** Западная, молодёжь 25–35, нет русскоязычного рынка.

---

### 2. Habitica (приложение + веб)
**Ссылка:** habitica.com
**Что умеет:** Трекер привычек в формате RPG-игры. Персонаж, квесты, гильдии, награды за выполнение задач.
**Хорошо:** Геймификация на максимум, активное комьюнити, бесплатная основа.
**Плохо:** Сложно для новичков, нет AI, нет Telegram-интеграции. Дизайн устаревший.
**Аудитория:** Геймеры, молодёжь, западная аудитория.

---

### 3. @ai_habit_bot (Telegram-боты, общий тип)
**Ссылка:** ищутся через поиск в Telegram по "habit bot", "трекер привычек бот"
**Что умеет:** Базовый трекинг привычек через Telegram. Напоминания, отметки выполнения.
**Хорошо:** Прямо в Telegram — нет лишних шагов.
**Плохо:** Нет AI, нет красивого UX, боты заброшены, нет воронки из Instagram. Часто не работают.
**Аудитория:** Русскоязычные пользователи Telegram.

---

### 4. ChatGPT + Telegram (DIY-решения)
**Ссылка:** нет единого продукта, разные реализации
**Что умеет:** Пользователи создают личных AI-ассистентов через обёртку GPT в Telegram. Умеют почти всё, но без структуры.
**Хорошо:** Максимальная гибкость, AI отвечает на всё.
**Плохо:** Нет системы, нет трекинга, нет онбординга. Пользователь должен сам знать, что спрашивать.
**Аудитория:** Технически продвинутые пользователи.

---

### 5. Done (приложение, iOS/Android)
**Ссылка:** done.app
**Что умеет:** Минималистичный трекер привычек. До 3 привычек бесплатно, подписка для большего.
**Хорошо:** Очень простой интерфейс, работает без лишнего. Хорошая визуализация стриков.
**Плохо:** Нет AI, нет Telegram, нет русскоязычного продвижения, платная расширенная версия дорогая.
**Аудитория:** Минималисты, западная аудитория.

---

**Вывод по конкурентам:** Прямых конкурентов с форматом Instagram → Telegram AI-агент нет. Это незанятая ниша. Существующие решения — либо приложения без Telegram, либо Telegram-боты без AI и без воронки из Instagram.

## 3. UX и пользовательские сценарии

### Сценарий 1: Первое знакомство

**Путь пользователя:**
1. Видит пост в Instagram — "Трекер привычек прямо в Telegram"
2. Тапает ссылку в bio → Telegram открывается с ботом
3. Нажимает Start → бот знает контекст (откуда пришёл)
4. Видит 2 строки приветствия + одну кнопку "Начать"
5. Добавляет первую привычку за 30 секунд
6. Получает первое напоминание — "зацеплен"

**Типичные ошибки разработчиков:**
- Стена текста в приветствии — пользователь уходит
- Нет связи с Instagram-постом — разрыв ожидания
- Анкета до демонстрации ценности — слишком рано
- Нет "быстрой победы" за первую минуту

**Как сделать правильно:**
- Контекстный старт: deep link `?start=insta_habits` → бот сразу в теме привычек
- Приветствие: максимум 2 строки + 1 кнопка
- Первое действие = первая ценность (добавил привычку → видишь трекер)

---

### Сценарий 2: Ежедневное использование

**Путь пользователя:**
1. В 9:00 уведомление: "Отметь привычки на сегодня"
2. Тапает → Telegram открывается прямо в боте
3. Список привычек с кнопками ✅/❌ — отмечает за 10 секунд
4. Бот отвечает: "3/4 привычки, серия 7 дней 🔥"
5. Закрывает Telegram

**Типичные ошибки:**
- Спам: 5 уведомлений в день → мьют → забытие
- Нет памяти контекста — бот кажется тупым
- Сломанные кнопки после обновления
- Нет наград за прогресс — мотивация падает

**Как сделать правильно:**
- Одно уведомление в день в выбранное пользователем время
- AI-контекст: "вчера пропустил пробежку — как сегодня?"
- Стрики + бейджи за 7/30/100 дней

---

### Сценарий 3: Потеря интереса / отток

**Почему уходят:**
- Сломанный стрик = "зря старался" → потеря мотивации
- Бот не адаптируется к расписанию
- Нет социального аспекта / комьюнити
- Технические сбои (бот не отвечает)
- Надоедливые уведомления → мьют → забытие
- Нет новизны через месяц

**Как удержать:**
- Защита стрика: "Есть 1 пропуск без штрафа в месяц — использовать?"
- Win-back через 3 дня молчания: "Давно тебя не видели"
- Недельный отчёт в воскресенье — человек видит прогресс
- Реактивация через Instagram (посты для "потерявшихся")

---

## 5. Монетизация

### Модели монетизации Telegram-ботов

**Freemium** — самая популярная модель
- Базовый функционал бесплатно, платные фичи закрыты
- Конверсия в платных: 2–5% (широкая аудитория), 8–15% (нишевые боты)
- Пример: 5 запросов/день бесплатно, безлимит — платно

**Подписка**

| Тариф | Цена/мес |
|-------|----------|
| Базовый | 299–499₽ |
| Стандарт | 699–1300₽ |
| Pro | 1500–4500₽ |

Минимальная психологически воспринимаемая цена для РФ: **299₽/мес**

**Telegram Stars** (встроенная платёжка)
- Комиссия 30% берёт Telegram
- Плюс: не нужен платёжный шлюз, нет настроек
- Реальные кейсы: тысячи долларов/мес у работающих ботов

**Кредиты/токены**
- Низкий барьер: купил 100 кредитов за 99₽
- Минус: сложнее прогнозировать выручку

---

### Воронка Instagram → Telegram → оплата

```
10 000 подписчиков Instagram
→ 3–8% переходят в Telegram = 300–800 человек
→ 5–12% из них покупают подписку = 15–96 платящих
→ При 399₽/мес = 6 000–38 000₽/мес MRR
```

---

### Что работает для маленьких проектов (до 1000 пользователей)

1. **Organic Instagram** — 3–5 Reels/неделю с демонстрацией бота, ноль бюджета
2. **Посевы в нишевых Telegram-каналах** — 500–5000₽ за пост, считать ROI
3. **Взаимный пиар** — договориться с 2–3 каналами в нише
4. **Lifetime Deal на старте** — сниженная цена для первых 50–100 платящих
5. **Telegram Stars** — самая простая платёжка без настроек

---

### Что НЕ работает на старте

- Google/Meta Ads — дорого, без конверсии убыточно
- TelegramAds — минимальный бюджет €1500
- Монетизация без аудитории — сначала 100+ активных пользователей

---

### Типичные ошибки при монетизации

1. Слишком дёшево или дорого без тестов — ставь цену, тестируй 2 недели, меняй
2. Нет бесплатного уровня — без freemium пользователь не "подсаживается"
3. Монетизация с первого дня без аудитории
4. Сложный онбординг — ушёл если не понял за 30 секунд
5. Нет напоминаний про апгрейд при исчерпании лимита
6. Платёжка с трением — никаких "переводов на карту вручную"

---

### Рекомендуемая схема монетизации для InstaTG на старте

1. **Freemium** — 5 бесплатных запросов в день
2. **Один тариф** — 399₽/мес через Telegram Stars
3. **Трафик** — 3–4 Reels в неделю + взаимный пиар в 2–3 нишевых каналах
4. **Retention** — напоминание при исчерпании лимита + еженедельный отчёт

**Цель первого месяца:** 20–50 платящих пользователей = 8 000–20 000₽/мес MRR
