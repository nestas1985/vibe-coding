> Исследование проведено: июнь 2026

## 1. Что такое AI-агент и чем он отличается от бота

**Бот** — это скрипт. Работает по заранее написанным сценариям. Спросил не так — не понял.

**AI-агент** — это система с «мозгом» (языковой моделью вроде GPT или Claude), которая:
- **понимает** смысл задачи, а не только ключевые слова
- **принимает решения** сама, без жёсткого сценария
- **действует**: открывает сайты, нажимает кнопки, скачивает файлы, отправляет сообщения
- **помнит** контекст и адаптируется

> Бот отвечает на вопрос. Агент — **решает проблему**.

## 2. Где живёт агент — на компьютере или в интернете?

### Локальный агент (на компьютере пользователя)
- Запускается локально, работает пока компьютер включён
- **Когда закрыл ноутбук — агент остановился**
- Плюсы: приватность, бесплатно, полный контроль
- Примеры: LocalGPT, Jan.ai, Ollama + агент

### Облачный агент (на серверах в интернете)
- Живёт на серверах провайдера (OpenAI, Anthropic, n8n cloud и др.)
- **Работает 24/7, даже когда ты спишь**
- Плюсы: автономность, не зависит от твоего железа
- Примеры: Make.com, Zapier Agents, n8n cloud, OpenAI Assistants

### Гибридный вариант
- Логика агента — в облаке, но он может управлять твоим компьютером через специальные программы (computer use)

## 3. Что агент умеет делать

Реальный сценарий — полная цепочка:

```
[Сайт 1] → скачать файл
    ↓
[Агент] → прочитать, обработать, сделать выжимку
    ↓
[Сайт 2] → загрузить файл
    ↓
[Telegram] → отправить сообщение с итогом
```

### Полный список возможностей агента:
- Открывать сайты и кликать по кнопкам (web browsing)
- Скачивать и загружать файлы
- Читать и писать письма (Gmail, Outlook)
- Постить в соцсети (Instagram, Twitter/X, VK)
- Писать в мессенджеры (Telegram, WhatsApp, Slack)
- Работать с Google Docs, Sheets, Drive
- Запускать код
- Искать информацию и делать выжимки
- Создавать события в календаре
- Делать звонки (через Twilio и подобные)

## 4. Классификация агентов

| Тип | Что делает | Примеры |
|-----|-----------|---------|
| **LLM-агент** | «Мозг» на языковой модели, понимает и рассуждает | ChatGPT, Claude, Gemini |
| **RPA-агент** | Роботизирует рутину: кликает, копирует, заполняет | UiPath, Automation Anywhere |
| **Веб-агент** | Ходит по сайтам, собирает данные, нажимает кнопки | Manus, Operator (OpenAI) |
| **Автономный агент** | Сам ставит подзадачи и выполняет цепочку действий | AutoGPT, Devin |
| **Мультиагент** | Несколько агентов работают в команде | CrewAI, AutoGen, LangGraph |
| **Workflow-агент** | Связывает сервисы в цепочки без кода | n8n, Make, Zapier |
| **Компьютерный агент** | Управляет рабочим столом как человек | Claude Computer Use |
| **Голосовой агент** | Понимает и говорит голосом | ElevenLabs + LLM, Vapi |

## 5. Как агент связывается с внешним миром

Агент общается с миром через **инструменты (tools)**:

```
           ┌─────────────────────────────────┐
           │           AI-АГЕНТ              │
           │   (мозг: GPT / Claude / Gemini) │
           └────────────┬────────────────────┘
                        │ использует инструменты
        ┌───────────────┼───────────────────┐
        ▼               ▼                   ▼
  [Интернет]      [Компьютер]         [Сервисы]
  • поиск         • файлы             • Telegram
  • сайты         • код               • Gmail
  • API           • приложения        • Google Drive
  • парсинг       • desktop           • Notion
                                      • Slack
                                      • Instagram
```

**MCP (Model Context Protocol)** — новый стандарт 2025 года, который позволяет любому агенту подключаться к любому сервису по единому принципу. Разработан Anthropic.

## 6. Главные инструменты и платформы

### Без кода (для начинающих)

| Платформа | Что умеет | Интеграций | Цена |
|-----------|-----------|-----------|------|
| **Zapier** | Самый простой, много готовых связок | 8 000+ | от $0 |
| **Make.com** | Визуальный конструктор сценариев | 1 500+ | от $0 |
| **n8n** | Мощный, с полным контролем, есть self-hosted | 500+ | от $0 |

### Фреймворки (для разработчиков)

| Фреймворк | Для чего |
|-----------|---------|
| **LangChain** | Универсальная основа для LLM-агентов |
| **CrewAI** | Команды агентов с ролями (как отдел сотрудников) |
| **AutoGen** (Microsoft) | Агенты-коллеги, работают вместе |
| **LangGraph** | Сложные сценарии с циклами и памятью |
| **Claude Agent SDK** | Агенты на базе Claude (Anthropic), 2026 |

### Готовые автономные агенты

| Агент | Что делает |
|-------|-----------|
| **Manus** | Автономный агент, работает в браузере, делает задачи сам |
| **Operator** (OpenAI) | Ходит по сайтам, делает покупки, заполняет формы |
| **Devin** | Пишет и деплоит код автономно |
| **AutoGPT** | Сам ставит и выполняет цепочку подзадач |

## 7. Мультиагентные системы

Несколько агентов могут работать **командой**, как отдел сотрудников:

```
[Менеджер-агент]
      ├── [Агент-исследователь] → ищет информацию
      ├── [Агент-аналитик]     → обрабатывает данные
      ├── [Агент-писатель]     → составляет текст
      └── [Агент-отправщик]   → публикует результат
```

Каждый агент специализируется на своей задаче. Менеджер координирует.

## 8. Статистика рынка (2025-2026)

- 62% организаций экспериментируют с AI-агентами (McKinsey, 2025)
- 23% уже масштабируют агентные системы в работе
- Telegram стал одной из главных платформ для запуска агентов

## Главные выводы

**Агент = мозг + инструменты + память**

| Хочешь | Решение |
|--------|---------|
| Работает 24/7 без тебя | Облачный агент (n8n cloud, Make, Zapier) |
| Приватно и бесплатно | Локальный агент (только пока ноутбук включён) |
| Связка сайт→файл→Telegram | n8n или Make, делается без кода за пару часов |
| Команда агентов | CrewAI или AutoGen |
| Максимум автономии | Manus или Operator |
